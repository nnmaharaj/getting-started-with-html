Web basics

The World Wide Web is composed of html documents that are linked together via hyper links. Servers serve up html pages that travel through the internet and to the user's browser. HTML stands for HyperText Markup Language. HTPP is the protocol via which web pages served by servers travel through the internet. The browser interprets the html document into human legible prose and style.

What language does your browser speak?.

Tags! They are everywhere...in HTML that is. They have contents in the middle, and are flanked on both sides by the tag name. The left side of the tag is called the opening tag while the right side of the tag is called the closing tag. The closing tag typically has a forward slash before the tag name. Examples include:
1. The b tag aka the bold tag.
2. The em tag aka the italics or emphasis tag.

Attributes! Attributes equal a value in the opening tag. Examples include:
1. The anchor tag, a href="www.youdon'tsay.com".
2. The image tag, img src="source.url.com" alt="text". It is known as a void tag because it has no content. Without content there is no need for a forward slash in the closing tag.

Whitespace! The following tags control white space:
1. The line break tag, br, also a void tag. Works as an in-line element.
2. The paragraph tag, p. Works as a block element.
3. The span element, span. It is an in-line element. Styles can be added to this element.
4. The div element, div. It is a block element. Styles can be added to this element.
